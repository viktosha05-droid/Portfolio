# Global Tech Market Statistical Research

## 📌 Project Overview
Цей проєкт присвячений статистичному дослідженню глобального ринку праці в ІТ на основі опитування понад 49,000 респондентів. Основна мета — виявити кореляції між досвідом, освітою та рівнем компенсації, а також проаналізувати тренди віддаленої роботи.

## 🎯 Business Goals
- Провести очищення та підготовку масивних даних (Data Cleaning).
- Визначити ключові показники компенсації для різних сегментів ринку.
- Знайти найбільш прибуткові індустрії для фахівців, що працюють віддалено.
- Візуалізувати розподіл доходів для кращого розуміння ринкових трендів.

## 🛠 Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
- **Environment:** Jupyter Notebook / Anaconda Cloud

## 🚀 Key Steps & Analysis
1. **Data Preprocessing:** Обробка пропущених значень та виправлення типів даних.
2. **Exploratory Data Analysis (EDA):** Розрахунок медіани, моди та 75-го перцентиля (benchmarks) для різних категорій розробників.
3. **Statistical Analysis:** Порівняння рівнів зарплат залежно від типу зайнятості (Remote vs On-site).
4. **Data Visualization:** Побудова графіків розподілу компенсацій та популярності індустрій.
