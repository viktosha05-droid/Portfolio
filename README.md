# Global Tech Market Statistical Research

## 📌 Project Overview
Этот проект посвящен статистическому исследованию глобального рынка труда в ИТ на основе опроса более 49 000 респондентов. Основная цель — выявить корреляции между опытом, образованием и уровнем компенсации.

## 🎯 Business Goals
- Провести очистку и подготовку данных (Data Cleaning).
- Определить ключевые показатели компенсации для различных сегментов рынка.
- Найти наиболее прибыльные индустрии для специалистов на удаленке.
- Визуализировать распределение доходов.

## 🛠 Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn

## 🚀 Key Steps & Analysis
1. **Data Preprocessing:** Обработка пропущенных значений.
2. **Exploratory Data Analysis (EDA):** Расчет медианы и 75-го перцентиля (benchmarks).
3. **Statistical Analysis:** Сравнение уровней зарплат (Remote vs On-site).
4. **Data Visualization:** Построение графиков распределения.

## 📈 Results
- Рассчитан 75-й перцентиль заработных плат для определения границ высокооплачиваемых ролей.
- Выявлены топ-3 индустрии с самой высокой оплатой для удаленной работы.
